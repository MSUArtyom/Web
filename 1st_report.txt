Биллинговая база оператора связи



Описание страниц

С любой страницы можно перейти на главную.

Главная страница
	Ссылка на список клиентов
	Ссылка на список услуг
	Форма для поиска клиента по имени/названию, отсылает на персональную страницу клиента
	Форма для поиска услуги по названию, отсылает на страницу услуги
	Ссылка на регистрацию транзакций
	Ссылка на регистрацию договора
	
Страница "Список клиентов"
	Список всех клиентов
	На каждого клиента можно нажать, чтобы перейти на его страницу
	Сортировка/поиск клиентов по оказываемым услугам в заданном интервале времени, по характеристикам их счетов, по имени
	Форма для добавления нового клиента в БД

Страница клиента
	Информация о клиенте
	Возможность редактирования
	Ссылка на историю конкретного счета данного клиента
	Ссылка на историю услуг данного клиента
	Ссылка на регистрацию новой денежной транзакции
	Ссылка на регистрацию нового договора
	Форма для добавления нового счета для этого клиента в БД

Страница истории счета
	История транзакций связанных с данным счетом клиента за заданный интервал времени
	Ссылка на клиента
	Ссылка на регистрацию новой денежной транзакции, связанной с данным счетом
	Ссылка на регистрацию нового договора, связанного с данным счетом

Страница истории услуг
	История услуг данного клиента
	Ссылка на клиента
	На каждую услугу можно нажать, чтобы перейти на её страницу

Страница "Список услуг"
	Список всех услуг
	На каждую услугу можно нажать, чтобы перейти на её страницу
	Сортировка услуг по типу, по имени
	Форма для добавления новой услуги в БД

Страница услуги
	Информация о услуге
	Возможность редактирования
	Ссылка на регистрацию нового договора, связанного с данной услугой

Страница регистрации транзакции
	Форма для регистрации транзакции. Указывается валютное значение, 
тип (списание или поступление) и номер счета. В случае списания указывается номер договора.

Страница регистрации договора
	Форма для регистрации договора. Указывается услуга, номер счета, период.




Сценарии использования

Добавление клиента в БД
	Перейти на главную страницу
	Перейти на страницу списка клиентов
	Заполнить необходимую форму, подтвердить изменения

Добавление счета клиента в БД
	Перейти на главную страницу
	Ввести в нужное поле имя клиента
	Перейти на страницу клиента
	Заполнить необходимую форму, подтвердить изменения

Добавление услуги в БД
	Перейти на главную страницу
	Перейти на страницу списка услуг
	Заполнить необходимую форму, подтвердить изменения

Регистрация договора
	Перейти на главную страницу
	Перейти на страницу регистрации договора
	Заполнить необходимые поля, подтвердить

Регистрация транзакции
	Перейти на главную страницу
	Перейти на страницу регистрации транзакции
	Заполнить необходимые поля, подтвердить
